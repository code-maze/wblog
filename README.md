# wblog
a blog for us

## 依赖第三方库

- python  
  1. Flask                --> python的微型web框架  
  2. Flask-RESTful        --> 用于编写RESTful的api  
  3. Flask-SQLAlchemy     --> 用于操作数据库的ORM框架  
  4. Jinja2               --> html模板解析生成器  
  
- css  
  1. Bootstrap          --> 响应式CSS框架  
  2. Font-Awesome       --> 字体图标库  
  
- js  
  1. JQuery             --> js的常用库  
  2. showdown.js        --> 用于解析markdown语法  
  3. highlight.js       --> 用于代码高亮
